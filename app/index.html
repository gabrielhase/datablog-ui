Demo Mode  Try the very first features the world has ever seen of Livingdocs.

 Write your text inline and press enter to get more paragraphs.

 Long-press on any element on the page to drag it around.

 Press the Plus sign at the right to see all available page elements and simply drag them to the page where you want them.

  Currently only the map element has properties. Use them to connect different demo data source from Zurich's Open Data portal.

  This is a demo site! We don't save any of your data, this is just to play around.
